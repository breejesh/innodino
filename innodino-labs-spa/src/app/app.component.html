About InnoDino Labs

We believe that learning to code should be as natural and joyful as playing with toys. Our mission is to introduce young minds to computational thinking through engaging, hands-on experiences that spark curiosity and build confidence.

Developed at Georgia Tech's Educational Technology program, InnoDino combines cutting-edge pedagogical research with playful, dinosaur-themed design to create an educational experience that kids love and parents trust.

Our Educational Modules

DinoBot Module

Motorized Robot + Ultrasonic Sensor

Build and program your own autonomous robot! Watch your code come to life as you control motors, read sensors, and create intelligent behaviors that respond to the environment.

LED Matrix Module

8x8 LED Display + Ultrasonic Sensor

Create stunning light shows and interactive games! Program individual LEDs to display patterns, animations, and respond to sensor input for immersive visual experiences.
